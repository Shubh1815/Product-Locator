# Product Locator

## Setting Up

## Backend
```
  $ pip install pipenv
```

#### Installing Dependencies
```
  $ pipenv install
```

#### Migrating to database
```
  $ pipenv shell 
  $ python manage.py migrate
```

#### Starting Server
```
  $ pipenv shell
  $ python manage.py runserver
```

## Frontend

#### Installing Dependencies

```
  $ npm install
```

#### Starting Server

```
  $ npm start
```

## Creating new admin
```
  $ python manage.py createsuperuser
```
