# Product Locator

## Setting Up

## Backend

```
  $ pip install pipenv
```

#### Installing Dependencies

```
  $ cd backend
  $ pipenv install
```
#### Starting Server

```
  $ pipenv shell
  $ python manage.py runserver
```

## Frontend

#### Installing Dependencies

```
  $ cd frontend 
  $ npm install
```

#### Starting Server

```
  $ npm start
```
